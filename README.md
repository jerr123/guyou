# 菇友·基于codeigniter2 的一个社交网站

## 声明

* 本项目仅用于学习
* 并非最新版本，存在许多未修复 bug 以及一些未完成的功能
* 开发于 很久很久以前(大二?)
* 整理于 2018/12/21 日

## 版本要求
* PHP5.6
* Mysql5.7
* Apache2.*

## 安装
* 导数 `guyou.sql` 数据库文件
* 配置 application/config/database.php
* 配置 Apache 服务器 vhost 到入口文件
* uploads 目录写入权限
* xx.com——主页
* xx.com/admin  后台，admin/admin
